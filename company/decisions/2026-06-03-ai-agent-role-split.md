# 2026-06-03 AIエージェント役割分担

## 決定
Claude Code と Codex を MyLife / company 運用で並行利用する。

Claude Code は CEO / PM / 秘書寄りの判断、優先順位整理、会社運用、ドキュメント作成を主担当にする。Codex は開発、コードレビュー、技術検証、実装タスク、PR ベースの変更管理を主担当にする。

## 理由
Claude Code は会社側の文脈整理や判断ログと相性がよく、Codex は実装リポジトリの調査、修正、検証、レビューと相性がよい。役割を分けることで、事業判断と技術実行を同時に進めやすくなる。

## 運用ルール
- Claude Code が「何をやるか」「優先順位」「判断理由」を整理する。
- Codex が「どう実装するか」「どう検証するか」「レビュー結果」を具体化する。
- 両方が関わる案件は、重要な判断と未解決事項を `company/` に戻して引き継ぐ。
- 技術的に重要な判断は、必要に応じて `company/decisions/` に記録する。

## 次の行動
既存の company 運用ドキュメントへ、この役割分担を反映する。
