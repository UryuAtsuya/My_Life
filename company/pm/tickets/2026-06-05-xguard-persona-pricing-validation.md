# XGuard persona / pricing validation

## 内容

XGuard v0の初期ICPと価格仮説を、実際のヒアリング・LP文言・支払い意思確認へ落とす。

## 仮説

- 初期ICPは「Xが売上導線・告知導線・信用導線になっている人」。
- 優先セグメントは、X中心の個人クリエイター/個人事業者、仕事用Xアカウントを使う小規模店舗・サロン・講座販売者、凍結リスクが高い創作・成人向け/グレー領域クリエイター。
- 初期LPではセンシティブ属性を前面に出さず、「事前バックアップ」「本人性の証明」「再起動導線」「仕事用アカウント保険」と表現する。
- 価格は `Free診断 / 1,980円 Personal / 2,980円 Pro / 4,980円 Concierge検証枠` を検証する。

## 完了条件

- [ ] 各優先セグメントからヒアリング候補を3人ずつ仮リスト化する。
- [ ] 8問の支払い意思確認質問をGoogle Formまたはヒアリング台本にする。
- [ ] `Free診断 / Personal / Pro / Concierge` のLP文言案を作る。
- [ ] 「backupだけ」「proof page込み」「手動再起動支援込み」で支払い意思を分けて聞く。
- [ ] 自動投稿、自動DM、自動follow/unfollow、BAN回避訴求をLP文言から除外する。
- [ ] 調査結果を `company/projects/x-ban-recovery-storage/requirements/` に追記し、必要なら価格仮説を更新する。
